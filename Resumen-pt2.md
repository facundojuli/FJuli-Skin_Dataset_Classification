En esta segunda parte del trabajo se abordó el problema de clasificación de lesiones dermatológicas utilizando una red neuronal convolucional (CNN) implementada en PyTorch. Se tomó como punto de partida una arquitectura basada en bloques convolucionales inspirados en AlexNet, reemplazando el modelo MLP utilizado previamente por una CNN capaz de extraer automáticamente características espaciales de las imágenes.

Durante el desarrollo se incorporaron distintas técnicas para mejorar la capacidad de generalización del modelo. Se mantuvo el uso de Data Augmentation mediante Albumentations, evaluando diferentes combinaciones de transformaciones como Horizontal Flip, Vertical Flip y Random Brightness Contrast. Asimismo, se utilizaron técnicas de regularización como Dropout y se incorporaron capas de Batch Normalization luego de las capas convolucionales para estabilizar el entrenamiento y favorecer una convergencia más rápida.

También se realizó una búsqueda de hiperparámetros variando el tamaño de entrada de las imágenes, batch size, learning rate, probabilidad de Dropout y parámetros de Data Augmentation. Con el objetivo de mantener tiempos de entrenamiento compatibles con los recursos disponibles, se redujo el espacio de búsqueda descartando configuraciones poco representativas y se ajustaron parámetros como la cantidad máxima de épocas y la paciencia del Early Stopping, conservando las combinaciones de mayor interés. Además, se registraron los hiperparámetros, métricas y modelos obtenidos mediante MLflow y TensorBoard para facilitar la comparación entre experimentos.

Finalmente, se comparó el desempeño de las distintas configuraciones utilizando las métricas de accuracy y loss sobre los conjuntos de entrenamiento y validación, analizando el efecto de las técnicas de regularización y de los distintos hiperparámetros en la capacidad de generalización del modelo.

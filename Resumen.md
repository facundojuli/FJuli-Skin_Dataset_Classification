En este trabajo se desarrolló un pipeline de clasificación de imágenes utilizando una red neuronal MLP implementada en PyTorch. 

Primero se ejecutó 0_eda.ipynb, donde se realizó un análisis exploratorio del dataset inspeccionando clases, cantidad de imágenes, resoluciones y ejemplos visuales. Luego se ejecutó 1_modelo simple.ipynb, donde se entrenó un modelo MLP simple como baseline sin búsqueda de hiperparámetros, con el objetivo de validar el correcto funcionamiento del pipeline completo de carga de datos, transformaciones, entrenamiento, validación y logging de métricas. Se partió de un modelo base con imágenes aplanadas (Flatten) y capas lineales, evaluando su desempeño en un problema de clasificación. A lo largo del experimento se incorporaron distintas técnicas para mejorar la performance y reducir el overfitting, como Dropout, Batch Normalization, data augmentation con Albumentations y regularización L2 (weight decay). También se modificó el pipeline de entrenamiento para integrar herramientas de seguimiento como TensorBoard y MLflow, permitiendo registrar métricas, hiperparámetros y visualizaciones del entrenamiento. 

Se analizaron diferentes estrategias de inicialización de pesos (He, Xavier y uniforme), observando su impacto en la estabilidad y velocidad de convergencia. Además, se visualizó la evolución de los pesos mediante histogramas en TensorBoard para comparar distribuciones a lo largo del entrenamiento. Finalmente, se compararon los resultados en términos de accuracy y loss en entrenamiento y validación, observando cómo las técnicas de regularización mejoraron la generalización del modelo y redujeron el overfitting.
